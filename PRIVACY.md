# Privacy Policy for Tape

**Last updated:** June 20, 2026

## Data Collection

Tape does **not** collect, transmit, or share any personal data, usage statistics, or analytics.

All recorded scripts and extension state are stored **locally** in your browser's `chrome.storage.local` and never leave your device.

## Permissions

| Permission | Purpose |
|------------|---------|
| `storage` | Store recorded scripts and extension state locally in your browser |
| `activeTab` | Interact with the currently active tab to record and replay actions |
| `<all_urls>` | Work on any website the user chooses to record or replay on |

## Third-Party Services

Tape does not use any third-party analytics, tracking, or external services. The extension contains no remote code and makes no network requests.

## Changes

If this policy changes, the updated date at the top will be revised.
